# 🌸 Animes Yoritai

Meu primeiro projeto: um site sobre animes!

Este projeto é uma introdução ao tema, feito com o objetivo de praticar desenvolvimento web e explorar a cultura dos animes de forma criativa.

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript
- PHP
- Futuramente: bibliotecas e outras melhorias)

---

## 📄 Sobre o Projeto

O site apresenta uma introdução ao universo dos animes, com foco em estilo visual e organização de conteúdo. Em futuras versões, pretendo incluir:

- Catálogo de animes favoritos  
- Filtros por gênero  
- Interface mais interativa  
- Layout responsivo
